리녹스  파일 
ls ,ls -a : 폴더 보기(숨김파일 보기 옵션)
cd 폴더명 : 지정 폴더로 이동
cat 파일명 : 텍스트 파일 보기 명령
touch 파일명 : 파일이 새 파일이면 빈 파일 생성

git 명령어
init / add / commit / status / log
init -- 초기
git status -- 상태보기
add - stage에 올린다 --> git add "file"
commit - stage에서 큐 싸인 저장 --> git commig -m "version
git log -- hash code를 확인할 수 있다
git checkout "hash code"
git add . -- 모든 파일을 stage에 올린다
